Features
Everything in one place

End-to-end visual inspection pipeline — from uploading raw photos to running live inference — without leaving the app.

🗂️

Dataset Manager

Upload images, tag them, and organise Train / Val / Test splits. Bulk import and metadata tagging built in.

✏️

Annotation Canvas

Draw bounding boxes and polygons directly on images. Multi-class labelling with keyboard shortcuts for speed.

📦

Dataset Versions

Snapshot datasets with preprocessing and augmentation configs so every experiment is reproducible.

⚡

Training Pipeline

Configure and launch YOLOv8-style training runs from the UI. Watch mAP, precision, and recall update in real time.

🔬

Model Comparison

Browse all trained models side by side. Compare mAP / precision / recall / latency to pick the best one.

🧪

Live Inference

Load any trained model, feed it fresh images, and see detected defects overlaid with class labels and confidence scores.

Workflow
How a project flows

InspectX mirrors the real ML pipeline — structured so you always know where you are and what comes next.

01

Create a project & upload images

Start a new inspection project, define defect classes (e.g. "scratch", "dent", "burr"), and drag in your raw images.

02

Annotate with the canvas editor

Draw boxes or polygons around each defect. Switch classes from the sidebar — every label saves automatically.

03

Snapshot a dataset version

Lock in a version with your chosen augmentation and preprocessing settings before training begins.

04

Train & compare models

Launch a training run, watch metrics live, and compare results across multiple experiments.

05

Run inference on new images

Feed unseen images to your best model and review detected defects with confidence score overlays.
